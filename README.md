# Joyclick

Joyclick is a python script that makes your controller able to control your mouse and keyboard functions.

# Dependencies
Joyclick currently needs the following:
<ul>
  <li>Python3</li>
  <li>PyAutoGui</li>
  <li>Pygame</li>
</ul>
